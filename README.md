# Deep-Learning-Architectures-for-Medical-Imaging-and-Genomics

This repository contains a PyTorch implementation of deep learning architectures applied to two distinct biomedical challenges. The first task tackles medical image classification using the BloodMNIST dataset. We developed and optimized Convolutional Neural Networks (CNNs) to classify normal blood cells, experimenting with explicit Softmax layers and MaxPool2d integration. The models utilizing Maxpooling achieved the best performance by providing critical feature abstraction and translation invariance while significantly reducing computational costs and memory usage.

The second task frames genomic analysis as a regression problem to predict the continuous binding affinity of the RBFOX1 protein directly from RNA sequences. We implemented and compared a local motif-scanning 1D-CNN against global sequence models, specifically a Bidirectional LSTM (Bi-LSTM) with and without attention mechanisms, using Optuna for automated hyperparameter tuning. The attention-augmented Bi-LSTM significantly outperformed the baseline by successfully capturing the global structural context of the sequences. Finally, the project explores the theoretical design of a scalable, dual-branch "Two-Tower" architecture to fuse RNA sequence data and trainable protein embeddings, aiming to generalize binding predictions across multiple proteins.
